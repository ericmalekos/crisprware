# Tutorials

Interactive Colab notebooks that walk the pipeline with explanations, code, and output. The first cell
sets up the environment from GitHub, so they run end to end in the browser.

- **[Full tutorial](https://colab.research.google.com/github/ericmalekos/crisprware/blob/main/colab_notebooks/CRISPRware_Tutorial.ipynb)**
  -- all major CRISPRware functions.
- **[Rice genome](https://colab.research.google.com/github/ericmalekos/crisprware/blob/main/colab_notebooks/RiceGenome_Tutorial.ipynb)**
  -- end-to-end on the rice osa1_r7 genome + annotation.
- **[NGS applications](https://colab.research.google.com/github/ericmalekos/crisprware/blob/main/colab_notebooks/NGS_Tutorial.ipynb)**
  -- retrieving public NGS data for custom gRNA library design.
